# Moulinette Time


42 인트라(v3) 프로필에서 내 로그타임을 보여주는 브라우저 확장 프로그램입니다. 

<br>

> [!IMPORTANT]  
> 프랑스 시간과 한국 시간 차이로 인해 몇 분의 차이가 있을 수 있습니다. 참고하세요.

<br>

> [English](./README.eng.md)


<br>

## 뭘 보여주나요?

- **진행 상황**: 누적 시간 / 목표 시간, 진행 바, 부족 시간
- **하루 평균**: 오늘부터 매일 몇 시간씩 하면 목표를 채우는지
- **주차별 / 월별 합계**: 주 단위(일~토), 월 단위로 잘라서 집계
- **주 목표 모드**: "매주 40시간" 같은 주 단위 요건이 있는 분들을 위한 모드
  - 달성한 주는 ✓, 놓친 주는 ✕
  - (예: 경북대 현장실습).

- **제외 요일/날짜**: 일요일은 쉰다? 시험 날은 뺀다? 하루 평균 계산에서 제외 가능
- **한/영 전환**, 접기 버튼, 설정은 바꾸는 즉시 저장

<br>

## 설치 방법 (Chrome / Edge / Whale 등 크로미움 브라우저)

스토어에 올라가 있지 않아서 "개발자 모드"로 직접 로드해야 하는데, 1분이면 됩니다.

1. 이 저장소를 다운로드하세요 (초록색 **Code** 버튼 → **Download ZIP** → 압축 해제, 또는 `git clone`)
2. 브라우저 주소창에 `chrome://extensions` 입력 (Edge는 `edge://extensions`)
3. 우측 상단(Edge는 좌측)의 **개발자 모드** 스위치 켜기
4. **압축해제된 확장 프로그램을 로드합니다** 클릭 → 압축 푼 폴더 선택
5. [profile-v3.intra.42.fr](https://profile-v3.intra.42.fr) 에 로그인 상태로 접속

우측 하단에 패널이 뜨면 성공입니다. "데이터를 기다리는 중"이라면 로그타임 캘린더가 보이도록 스크롤해 보세요.

<br>

## 사용법

- **⚙ 설정**: 기간(시작/종료일), 목표(h), 주 목표(h), 제외 요일/날짜를 바꿀 수 있어요. 저장 버튼 없이 바꾸는 순간 적용 + 저장됩니다.
- **목표(h)**: 라피신이면 120을 넣으세요. 비워두면 목표 없이 기록만 보여줍니다.
- **주 목표(h)**: 주 단위 요건이 있으면 여기에 (예: 40). 목표(h)를 비우고 주 목표만  넣으면 상단 진행 바가 "이번 주" 기준으로 바뀌고 매주 리셋됩니다.
- **한/EN**: 헤더 버튼으로 언어 전환
- **−**: 패널 접기 (접어도 누적/목표 요약은 보여요)

<br>

## 어떻게 동작하나요?

인트라 프로필 페이지가 로그타임 캘린더를 그리려고 불러오는 데이터를
확장 프로그램이 감지해서 그대로 집계합니다. 별도 API 키나 로그인이 필요 없고,
데이터는 브라우저 밖으로 나가지 않아요. 설정은 `chrome.storage.sync`에 저장됩니다.
